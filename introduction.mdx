## Overview

Turntable has three components:

1. **Vinyl** - a python framework for building and managing analytics infrastructure.
2. **AI Agent (coming soon)** - an AI data analyst assistant that works with Vinyl to help you analysis and surface insights in record time.
3. **Turntable Cloud** - managed CPU and GPU compute to help you run intensive memory-heavy workloads.
