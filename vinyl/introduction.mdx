## Overview

Turntable has 4 components:

1.  **Sources** - We support 500+ data sources, including custom REST API and ODBC connections.
2.  **Models** - We use SQLMesh for writing data transformations using plain SQL.
3.  **Metrics** - A lightweight semantic layer built with simple YAML files
4.  **Charts & Dashboards** - Looker-like explores, charts and dashboards
